## Cocktail Drinks Folder Organization

Images are grouped by the drink's primary spirit or style to keep browsing and maintenance simple.

### Folders

- `gin/` - gin-forward classics (French 75, Negroni, Gimlet, Tom Collins)
- `tequila/` - tequila-based cocktails (Paloma, Margarita, Tequila Sunrise)
- `vodka/` - vodka-based cocktails (Moscow Mule, Vodka Martini, Bloody Mary, etc.)
- `rum/` - rum-based cocktails (Mojito, Dark 'n Stormy, Daiquiri)
- `whiskey_scotch/` - whiskey and scotch cocktails (Old Fashioned, Manhattan, Rob Roy, etc.)
- `sparkling_wine/` - sparkling wine and spritz-style drinks (Aperol Spritz, Mimosa)
- `novelty_soda/` - soda/novelty-style drinks (Dirty Root Beer, Cream Soda)

File names retain their original numeric prefix to preserve the previous ordering.
